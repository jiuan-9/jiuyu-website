## 编辑指引

### 如何添加一条新公告？

1. 在对应分类（重要公告 / 最新公告）下，复制一个完整的 `### 公告 N` 区块
2. 修改编号（确保序号连续）
3. 填写标题、日期、内容等信息
4. 保存文件，双击「推送公告.bat」完成推送

### 如何删除一条公告？

1. 删除整个 `### 公告 N` 区块（包括标题和内容）
2. 重新编号剩余公告（确保序号连续）
3. 保存文件，双击「推送公告.bat」完成推送

### 如何修改公告顺序？

1. 调整 `### 公告 N` 区块的上下位置
2. 重新编号（公告 1 显示在最前面）
3. 保存文件，双击「推送公告.bat」完成推送

### 格式说明

| 字段 | 是否必填 | 说明 |
|------|---------|------|
| 标题 | ✅ 是 | 公告标题，建议简洁明了，不超过 30 字 |
| 日期 | ✅ 是 | 格式：YYYY-MM-DD，例如：2026-07-18 |
| 标签 | ⭕ 可选 | 仅「重要公告」有此字段，如「重要更新」「紧急通知」等 |
| 内容 | ✅ 是 | 公告正文，可换行，支持多行文本 |

### 注意事项

- 重要公告建议不超过 3 条
- 最新公告建议不超过 5 条
- 请确保日期格式正确（年-月-日，如 2026-07-18）
- 内容中请勿使用特殊符号或 HTML 标签
