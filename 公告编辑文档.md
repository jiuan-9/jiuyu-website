# Quiddity 官网公告编辑文档

网站已移除 `/admin` 页面，公告管理请使用独立桌面管理后台。

## 使用独立管理后台（推荐）

1. 打开 `D:\quiddity-website\启动管理后台.bat`（或运行 `python main.py`）
2. 在左侧导航切换到 **Announcements**
3. 选择分类：
   - `important`：重要公告（建议 1-3 条）
   - `latest`：最新公告（建议 1-5 条）
4. 填写标题、日期、标签、内容
5. 点击「保存到 announcements.json」
6. 切换到 **Deploy** 页，输入 commit 信息，点击「一键推送」

## 填写规则

- 重要公告：填标题、日期、标签、内容
- 最新公告：填标题、日期、内容
- 标题或内容为空时，网站上不显示
- 日期格式：`年-月-日`，例如 `2026-07-18`
- 内容可以换行写
- 推送成功后，等待 1-3 分钟网站自动更新

## 注意事项

- 如需修改 GitHub Token 或项目根目录，进入 **Settings** 页
- 首次启动请先在 **Settings** 页确认项目根目录指向 `D:\quiddity-website`
